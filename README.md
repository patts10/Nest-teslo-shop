#Teslo API

1. Clone project
2. Run: ```npm install```
3.  Clone ```,env,template``` file and rename to ```.env```
4.  Set environment variables
5. Build DB
  ```
  docker-compose-up -d
  ```
6. Run: ```npm run start:dev```
